vue_Day01

Posted on 2020-09-16

1.0 项目搭建

​ 创建工程化的项目，一般会使用vue脚手架

​ npm i -g @vue/cli

1.1 创建项目（命令行）

​ vue create 项目名称

​

​ 下载完成之后，打开项目，在浏览器地址栏，输入 localhost:8080

​ cd 当前项目

​ `npm run serve

1.2 创建项目(UI)

​ vue ui

​ 根据页面提示创建即可，一般不要去使用

1.3 修改项目内容

​ 主要操作的内容在 src 文件夹

​ 比如配置文件，一保存就会报错，需要创建新的配置问价 .prettierrc

​ main.js 不用管，App.vue 是根组件，如果要显示内容，一定要保留 这个标签

​ views 里面一般写大页面，components 写的是一些小的公共组件

​ router 里面是路由规则，根据访问的路径，显示对应的组件，一定要先引入

​

2.0 组件化

​ 组件就是封装起来的具有独立功能的UI部件

​ 优点：可重用 可组合 易维护

3.0 表达式

​ 模板语法，Mustache 语法，就是俩花括号
